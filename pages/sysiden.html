Linear Time Invariant State Space System Identification Using Adam Optimization

In this paper, I have used Tensorflow's Adam optimizer to optimize state space matrices that makes a linear model that fits given sets of input and output variables. Some assumptions were made to the model and the technique was tested and proven to work even with high noise to signal ratios. Moreover, the technique successfully identified a system that violates some of the assumptions the model was built on, showing great potential in this technique.

The paper is on IEEEXplore and I have published the code for this paper and some examples on my GitHub.
